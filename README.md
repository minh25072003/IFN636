Video Playlist Manager
A simple full-stack web app that lets you manage video playlists — create, edit, and delete playlists, and add or remove videos inside them.

Features:
Create, edit, and delete playlists
Add, edit, and remove videos within a playlist
View all your playlists in one place

Setup

Clone the repo and navigate into the project
Run npm install in both the backend and frontend folders
Start each with npm start
Open http://localhost:3000 in your browser
